# Session 4: Module System + Project Structure + Runner

This folder contains the teaching package for Session 4.

## Main Topic

The main lesson for this session is:

- `import` / `export`
- named export vs default export
- splitting code by responsibility
- `types`, `utils`, `services`
- creating a runner so the project can be tested with one command

## Algorithm Add-On

This session also includes algorithm notes:

- `PREFIX_SUMS_GUIDE.md`
- `prefix_sums_examples.js`
- `TWO_POINTERS_GUIDE.md`
- `two_pointers_examples.js`

`Two Pointers` is included as the next algorithm after prefix sums.
