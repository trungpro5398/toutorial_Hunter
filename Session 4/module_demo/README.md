# Module Demo

This is a small TypeScript demo project for Session 4.

It shows:

- named exports
- default exports
- `types`, `utils`, `services`, `data`
- one runner file
- one-command execution

## Run

```bash
cd "/Users/quangtrungnguyen/Documents/Offer_Job/Tutorial/Session 4/module_demo"
npm run dev
```
