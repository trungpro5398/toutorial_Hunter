# Session 8 Effect Demo

Study planner demo for teaching `useState` and `useEffect`.

## What this demo shows

- controlled input with `useState`
- immutable add / toggle / delete updates
- `useEffect` to sync `document.title`
- `useEffect` to save tasks to localStorage
- `useEffect` with interval cleanup

## Run locally

```bash
npm install
npm run dev
```

Then open the local Vite URL in your browser.
